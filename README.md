# Text-to-Speech Web App

This is a simple web application that allows users to input text or upload a text file and have its contents read out using the browser's built-in text-to-speech functionality.

## Features

- Input text into a text area and have it read out using text-to-speech.
- Upload a text file and have its contents read out.
- Toggle between dark mode and light mode for better visibility.

## Getting Started

1. Clone the repository:

git clone https://github.com/Nagendraindus/filereader

2. Open `index.html` in a web browser.

3. Use the provided buttons to input text, upload a file, toggle modes, and control the reading function.
